API

Get The Most Recent Data

http://www.ti-fi-uofsc.com/$username/api/$devicename/get-data/

Fetches the latest data object that was stored on the server

Parameters

  • username – The username of the owner.

  • devicename – The name of the destination device.

Returns

HTTP Response 200.

{    "id": 0.0
     "Temperature": 0.0,
     "Humidity": 0.0,
     "Movement": 0.0,
     "Gas": 0.0,
     "Generic": 0.0,
     "Timestamp":"<ISO8601>"
 }

Post New Data

http://www.ti-fi-uofsc.com/$username/api/$devicename/post-data/

Creates a new data object and stores it in the database. The content of the data packet must follow specific format

Parameters

  • username – The username of the owner.

  • devicename – The name of the destination device.

  • body (JSON) – 

{
    "data": {
        "Temperature":0.0,
        "Humidity":0.0,
        "Movement":0.0,
        "Gas":0.0,
        "Generic":0.0
            },
    "timestamp": "<ISO8601>",
    ...
}
Returns

HTTP Response 201.

Get The Entire Device Data

http://www.ti-fi-uofsc.com/$username/api/$devicename/get-all-data/

Fetch the entire dataset stored under a given device

Parameters

  • username – The username of the owner.

  • devicename – The name of the destination device.

Returns

HTTP Response 200.

This project is under active development.
